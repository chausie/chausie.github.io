Hugo+GitHub Pages搭建远端博客

@Chausie · Jul 2, 2019 · 1 min read

一、Hugo安装、使用与配置 Hugo安装 下载并解压， Hugo（仅列举Windows） 我使用的版本是hugo_extended_0.55.6_Windows-64bit.zip。 解压后，还需再配置环境变量，在PATH中添加hugo.exe的路径,如：D:\hugo\ win+r输入cmd打开命令行提示符，输入并检验： $hugo version Hugo使用 一、 生成站点：使用Hugo快速生成站点，比如希望生成到 /path/路径：,创建文件夹名myblog的博客网站 $hugo new site /path/myblog 站点目录：网站根目录 · archetypes/ 内容类型，在创建新内容是自动生成内容的配置，存放default.md · data/ 存放自定义模版，导入的toml文件（或json，yaml） · content/ 网站内容，使用markdown格式 · layouts/ 网站的模板，决定内容如何呈现 · static/ 存放css，js，fonts，media等，觉得网站外观 config.toml 包括baseurl, title, copyright等等网站参数。 二、 创建文章: 创建第一篇文章，放到 post 目录，方便之后生成聚合页面。 $cd /path/ $hugo new post/first.md Hugo配置 一、 安装主题：没有主题是不行的。 到主题库挑一个心仪的主题，这里选择Hyde 创建themes目录 $cd themes $git clone https://github.com/spf13/hyde.git 二、 运行Hugo： hugo server --theme=hyde --buildDrafts 浏览器里访问:http://localhost:1313 二、部署到远端 前提：主机已经安装好git并有GitHub账号。你需要部署在 GitHub Pages 上，首先在GitHub上创建一个Repository，命名为：username.
